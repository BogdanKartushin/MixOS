Windows 3.11 for old games
Версия 0.95

        1. Что это такое?
    Windows 3.11 for old games, далее W311_OG — это сборка всем известного DOSBox и уже установленной системы Windows for Workgroups 3.11. 

        2. Для чего это нужно?
    Развитие компьютерных технологий не стоит на месте, и в этом, к сожалению, есть и отрицательные стороны, одной из которых является потеря совместимости со старыми программами. И если для абсолютного большинства программ есть современная замена, то с играми дело обстоит весьма плачевно — ведь не станет разработчик переписывать игрушку 15-летней давности из-за кучки фанатов, даже если это не такая уж и маленькая кучка; да и разработчик-то уже, возможно, давно не существует. Для решения этой проблемы создано множество эмуляторов, к которым относится и DOSBox.
    Игры для 16-битных версий Windows (3.0/3.1/3.11) являются отдельным, порою весьма тяжёлым случаем — хотя формально совместимость со старыми Windows до сих пор поддерживается, на практике шансы запустить подобную игру где-то 50/50, и во многих случаях это не лечится принципиально. Также надо заметить, что из 64-разрядных систем (которые стремительно набирают популярность) поддержка 16-разрядных программ вырезана полностью.
    Но не всё потеряно: Windows 3.x может работать в последних версиях DOSBox! Таким образом, вновь становится возможным поиграть в игры, которые на современных ОС нельзя запустить даже теоретически (например, Total Mayhem).
    Однако, для современного пользователя процесс установки и настройки Windows 3.11 может оказаться непростым занятием, особенно в среде DOSBox, который накладывает свои ограничения. Для того, чтобы преодолеть эти препятствия, и была создана данная сборка.

        3. Что внутри?
    Сборка состоит из следующих частей:
DOSBox, версия 0.73.
Microsoft Windows for Workgroups, версия 3.11 RUS.
Уже установлены следующие компоненты:
* Win32s;
* WinG;
* VfW (Video for Windows);
* драйверы:
    * Sound Blaster 16;
    * джойстика;
    * CD-Audio;
    * дисплея (S3 Trio 64, а также VESA, на всякий случай).
Все компоненты уже настроены и готовы к работе.

        4. Как пользоваться?
    Во-первых, необходимо умение пользоваться DOSBox (как минимум, знание основных параметров файла dosbox.conf и некоторых главных команд), иначе большинство игр вам установить не удастся.
    Во-вторых, необходимо хотя бы поверхностное знакомство с Windows 3.11 — хотя в целом интерфейс за прошедшие годы радикально не изменился, есть множество неочевидных моментов (например, отсутствие контекстного меню по нажатию правой кнопки мыши).
    Некоторые конкретные особенности:
* по умолчанию в качестве диска C: монтируется папка C_DRIVE, находящаяся в папке с программой, в неё же можно помещать и игры; менять её нельзя!
* при необходимости примонтировать другие папки желательно делать это в файле dosbox.conf в секции [autoexec], сразу после строки "mount c c_drive". Хотя работает и монтирование дисков прямо в работающей системе, делать это нежелательно (особенно для компакт-дисков, содержащих аудиодорожки).
* все операции с файлами можно выполнять через "Диспетчер Файлов", являющийся аналогом современного Проводника.
* переключение раскладки клавиш в Windows осуществляется сочетанием Левый Shift+Правый Shift.
* если не играет CD-Audio, необходимо сделать следующее: ПОСЛЕ монтирования привода зайти в "Панель управления", выбрать "Драйверы", из списка выбрать "[MCI] CD Audio" и нажать кнопку "Установка...". Должно появиться сообщение "Обнаружено одно устройство CDROM. Установка выполнена".
* после выхода из Windows можно запустить его снова, просто введя команду WIN.
